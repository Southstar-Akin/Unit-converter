How To Use This App

This converter is an easy to use tool with 3 preset measurements


  1. Length in Feet and Meters
  2. Volume in Gallons and Liters
  3. Mass in Kilograms and Pounds

The first input field, "Insert Units" takes the amount of units to be converted

Hitting the "Convert" buttons translates results into the section beneath

New measurements can also be added by "Double tapping" on the "Rate per Unit" box and following the prompts

After which units can be specified in the "Insert Units" box and Rate in the "Rate per Unit" box

NOTE: This Units Converter cannot save the new measurements you've specified as i have not implemented that feature yet
